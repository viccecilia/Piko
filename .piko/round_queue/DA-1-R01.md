# Round ID: DA-1-R01

Round Name: Article Candidate Contract

本轮目标:
Define a stable contract for converting a discovery cluster into an article candidate.

本轮任务:
- 执行任务:
  - Add or refine schema for an article candidate generated from `PlayerNeedCluster`.
  - Required fields should include game, player question, cluster id, decision, article intent, source query hints, risk flags, and safety flags.
  - Candidate must include `requires_evidence_pipeline=true` and `publish_ready=false`.
- 测试任务:
  - Add schema JSON round-trip tests.
  - Test publish_candidate cluster maps to a candidate.
- 协作验收任务:
  - Worker summary must include sample candidate JSON.

- `packages/shared/schemas.py`
