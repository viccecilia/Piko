# Round ID: DA-5-R02

Round Name: Candidate Pipeline API

本轮目标:
Expose a minimal internal API for discovery-to-article candidate flow.

本轮任务:
- 执行任务:
  - Add endpoint such as `POST /discovery/candidate/run`.
  - Endpoint should accept cluster id or selection filter and return artifact paths plus verification summary.
- 测试任务:
  - Test API returns safe artifact metadata.
  - Test blocked/high-risk cluster returns clear error or non-runnable status.
- 协作验收任务:
  - Worker summary must include request/response example.
