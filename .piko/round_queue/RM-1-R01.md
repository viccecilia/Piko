# Round ID: RM-1-R01

Round Name: Real Source Contract

本轮目标:

Define the real-market source contract for Steam, Reddit, JP community, KR community, and SERP/search snippets.

本轮任务:
- 执行任务:
  - Add or extend source contract documentation for real-market discovery.
  - Define required normalized fields for hot games and player questions.
  - Define per-source retained fields and prohibited fields.
  - Document source categories: `steam`, `reddit`, `jp_community`, `kr_community`, `serp_snippet`.
- 测试任务:
  - Add tests proving the source contract lists all required source categories and retained fields.
  - Add tests proving prohibited retention includes raw text, full posts, images, maps, credentials, and full copied tables.
- 协作验收任务:
  - Worker summary must include the contract location and a short retained/prohibited field table.
