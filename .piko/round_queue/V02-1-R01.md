# Round ID: V02-1-R01

Round Name: Human Approval Packet Contract

本轮目标:

建立把 GROW draft queue 转成可执行任务之前的人类批准包，确保任何执行都不是自动发生。

本轮任务:
- 执行任务:
  - 定义 `artifacts/v02_runtime/approval_packet_contract.json`。
  - 字段至少包含 `approval_id`、`source_draft_package`、`selected_task_ids`、`operator_decision`、`approval_scope`、`expires_at`、`risk_acknowledgement`、`status`。
  - 支持状态：`pending_operator_review`、`approved_for_materialization`、`rejected`、`expired`。
  - 明确没有 `approved_for_materialization` 时不得生成正式 round queue。
- 测试任务:
  - JSON parse probe。
  - 测试无 approval 时不能 materialize。
  - 测试 expired approval 不能 materialize。
- 协作验收任务:
  - 生成 `.piko/summaries/worker_V02-1-R01.md`。

允许修改:

- `packages/*`
- `tests/*`
- `docs/*`
- `artifacts/v02_runtime/*`
- `.piko/summaries/worker_V02-1-R01.md`
- `.piko/round_status.json`

禁止修改:

- 不要生成正式 `.piko/round_queue` 执行文件。
- 不要自动批准任何 draft。

必须运行的验证:

- Approval packet JSON parse probe。
- Approval required tests。

完成定义:

- 人类批准包合同存在，且没有批准时不会执行。

输出格式:
- 修改了什么
- 每个任务状态
- 验证结果
- 协作验收结果
- 未完成/风险
- 下一轮建议
