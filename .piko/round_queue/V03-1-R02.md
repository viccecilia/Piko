# Round ID: V03-1-R02

Round Name: Approval Scope And No-Install Policy

本轮目标:

为 first practical candidate 定义 approval scope、no-install policy、activation checklist，保证后续试点能推进但不会自动启用或污染 active runtime。

本轮任务:
- 执行任务:
  - 生成 approval scope artifact，说明本轮允许 dry-run / fixture / local adapter，不允许 install / network / LLM / active replacement。

  - 把该政策接入 V03 artifacts 或 docs。
- 测试任务:
  - 测试 approval artifact 中 risky action 默认 false。
  - 测试缺少 approval 时 materialization/activation 仍被阻断。
- 协作验收任务:
  - 生成 `.piko/summaries/worker_V03-1-R02.md` 和 `.piko/summaries/worker_V03-1.md`。
