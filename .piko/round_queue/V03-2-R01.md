# Round ID: V03-2-R01

Round Name: LangGraph-Style Adapter Contract

本轮目标:

建立 LangGraph-style workflow adapter contract，让 Piko 能以统一方式表达 nodes、edges、state、retry、Gate、verification handoff，而不依赖真实安装。

本轮任务:
- 执行任务:
  - 定义 adapter contract artifact 和必要代码结构。
  - 字段至少包含 graph_id、nodes、edges、state_schema、input_contract、output_contract、gate_policy、retry_policy、side_effect_policy、trace_policy。
  - 明确 real LangGraph package 是 optional backend，不是默认依赖。
- 测试任务:
  - Contract schema/JSON parse tests。
  - 缺少 required fields 时失败。
  - side_effect_policy 默认 disabled。
- 协作验收任务:
  - 生成 `.piko/summaries/worker_V03-2-R01.md`。

允许修改:

- `packages/*`
- `tests/*`
- `docs/*`
- `artifacts/v03_practical_plugin/*`
- `.piko/summaries/worker_V03-2-R01.md`
- `.piko/round_status.json`

禁止修改:

- 不 pip install。
- 不 import 必需外部 LangGraph。
- 不联网查询包信息。

必须运行的验证:

- Adapter contract tests

完成定义:

- Piko 有可验证的 LangGraph-style adapter contract。
- 缺真实 LangGraph 时仍可运行本地 fixture。

输出格式:
- 修改了什么
- 每个任务状态
- 验证结果
- 协作验收结果
- 未完成/风险
- 下一轮建议
