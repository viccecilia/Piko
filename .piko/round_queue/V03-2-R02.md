# Round ID: V03-2-R02

Round Name: Deterministic Local Graph Fixture

本轮目标:

实现一个本地 deterministic graph fixture，模拟成熟 workflow runtime 的 node 执行、state 传递、Gate 停顿和 trace 输出。

本轮任务:
- 执行任务:
  - 实现 local graph runner fixture。
  - 支持至少 4 个节点：source_discovery、evidence_rank、draft_handoff、verification_gate。
  - 输出 bounded trace artifact，不保存 raw source。
- 测试任务:
  - 测试 node 顺序、state 传递、Gate 输出、trace 可解析。
  - 测试 fixture 不触网、不调用 LLM。
- 协作验收任务:
  - 生成 `.piko/summaries/worker_V03-2-R02.md`。

允许修改:

- `packages/*`
- `tests/*`
- `artifacts/v03_practical_plugin/*`
- `.piko/summaries/worker_V03-2-R02.md`
- `.piko/round_status.json`

禁止修改:

- 不调用真实 connector。
- 不调用 article publish。

必须运行的验证:

- Local graph fixture tests
- Trace artifact parse probe

完成定义:

- 本地 graph fixture 可运行并输出 trace。
- 该 trace 能解释每个节点做了什么和为何停在 Gate。

输出格式:
- 修改了什么
- 每个任务状态
- 验证结果
- 协作验收结果
- 未完成/风险
- 下一轮建议
