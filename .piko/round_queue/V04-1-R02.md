# Round ID: V04-1-R02

Round Name: Explicit Pilot Approval Artifact

本轮目标:

生成 V04 pilot approval artifact。它只允许受控 dependency probe 和 smoke，不允许生产启用。

本轮任务:
- 执行任务:
  - 生成 explicit approval artifact。
  - 字段至少包含 pilot_approved、dependency_probe_allowed、install_allowed、network_allowed、llm_allowed、production_activation_allowed、approved_commands、rollback_note。
  - 若不允许安装，后续必须 blocked_for_approval。
- 测试任务:
  - 测试 production_activation_allowed=false。
  - 测试 approval 缺失或 false 时 install/probe 被阻断。
- 协作验收任务:
  - 生成 `.piko/summaries/worker_V04-1-R02.md` 和 `.piko/summaries/worker_V04-1.md`。

允许修改:

- `packages/*`
- `tests/*`
- `docs/*`
- `artifacts/v04_langgraph_backend/*`
- `.piko/summaries/worker_V04-1-R02.md`
- `.piko/summaries/worker_V04-1.md`
- `.piko/round_status.json`

禁止修改:

- 不批准 production activation。

- Approval artifact tests

完成定义:

- 后续 install/probe 是否允许有机器可读依据。
- 生产启用仍未批准。
