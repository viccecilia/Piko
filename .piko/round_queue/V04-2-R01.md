# Round ID: V04-2-R01

Round Name: Dependency Availability Probe

本轮目标:

在 approval artifact 允许的范围内探测 LangGraph 是否已经可用。若不可用，明确 blocked 状态，不伪装成功。

本轮任务:
- 执行任务:
  - 检查 approval artifact。
  - 若未批准 dependency_probe_allowed，输出 blocked_for_approval。
  - 若允许，尝试受控 import/version probe。
  - 生成 install/probe result artifact。
- 测试任务:
  - 测试 no approval -> blocked_for_approval。
  - 测试 unavailable dependency -> blocked_for_dependency。
  - 测试 probe artifact 不含 secrets/env dump。
- 协作验收任务:
  - 生成 `.piko/summaries/worker_V04-2-R01.md`。

允许修改:

- `packages/*`
- `tests/*`
- `artifacts/v04_langgraph_backend/*`
- `.piko/summaries/worker_V04-2-R01.md`
- `.piko/round_status.json`

禁止修改:

- 不自动安装，除非 approval 明确 install_allowed=true 且命令受控。
- 不输出完整环境变量。

必须运行的验证:

- Dependency probe tests

完成定义:

- LangGraph 当前可用性被真实记录。
- 不可用时正确 blocked。

输出格式:
