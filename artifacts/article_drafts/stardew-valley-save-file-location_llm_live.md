# Stardew Valley Save File Location

If your Stardew Valley save is missing, do not reinstall or edit files first. Check the save folder for your platform, then make a backup before changing anything.

## Quick Answer
Check the platform-specific Saves folder first, then back it up before editing or moving any files.

## Save File Locations
- **windows**: `%AppData%\StardewValley\Saves\`. Stardew Valley save files are stored at %AppData%\StardewValley\Saves\ on windows.
- **mac**: `~/.config/StardewValley/Saves/`. Stardew Valley save files are stored at ~/.config/StardewValley/Saves/ on mac.
- **linux**: `$XDG_CONFIG_HOME or ~/.config/StardewValley/Saves/`. Stardew Valley save files are stored at $XDG_CONFIG_HOME or ~/.config/StardewValley/Saves/ on linux.

## What To Check First
1. Check the platform-specific save folder first.
Sources: pcgamingwiki_31535
2. Back up the whole Saves folder before moving, deleting, or editing files.
Sources: pcgamingwiki_31535

## Do Not Try First
- Do not download unknown recovery tools, DLL files, or save patches.
- Do not delete the original save folder without a backup.
- Do not overwrite cloud and local saves until you know which one is newer.
