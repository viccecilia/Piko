# Stardew Valley Save File Location - Comparison

## Summary
Piko's draft is clearer as a player-facing checklist, but it still needs one more source-verification round before publishing.

## Materials Checked
- **PCGamingWiki - Stardew Valley**
  - URL: https://www.pcgamingwiki.com/wiki/Stardew_Valley#Save_game_data_location
  - Source type: reference_wiki
  - What it covers: Structured save game data locations for Windows, macOS, and Linux.
  - Notes: Strong structured source, but the raw wiki format is not written as a player-first troubleshooting checklist.
- **Stardew Valley Wiki - Saves**
  - URL: https://stardewvalleywiki.com/Saves
  - Source type: community_official_wiki
  - What it covers: Save transfer and troubleshooting guidance, including missing-save recovery context.
  - Notes: Useful for recovery context, but long and broad for a player who only needs the local save path first.
- **Steam Community discussion - save file location**
  - URL: https://steamcommunity.com/app/413150/discussions/0/405694031551275585
  - Source type: community_discussion
  - What it covers: Short player answers pointing to %AppData%/StardewValley/Saves on Windows.
  - Notes: Direct and practical, but less complete and less structured than a source-backed guide.
