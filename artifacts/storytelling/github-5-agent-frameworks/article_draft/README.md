# 图文版本草稿

这是《我让 Codex 每天扫描 GitHub 高星 AI 项目，最后发现 Agent 真正该抄的不是框架，而是这 5 个能力》的图文产出物。

## 文件

- `index.html`：长图文/公众号风格 HTML，可直接在浏览器打开预览。

## 使用方式

1. 打开 `index.html` 检查排版。
2. 使用浏览器整页截图或导出为图片/PDF。
3. 发布前如要写具体 GitHub star 数，先重新核验最新数据。

## 设计说明

- 参考用户提供的图文图例：白底、窄正文、段落解释、穿插大图和步骤说明。
- 不使用未经许可的外部图片。
- 所有配图均由 HTML/CSS 生成，包括 GitHub 扫描界面、能力地图、选题队列和内容包预览。
