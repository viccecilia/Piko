## Scene 01, 0-4s

最后发现，Agent 真正该抄的不是框架，而是 5 个能力。

## Scene 02, 4-10s

## Scene 03, 10-16s

我看完一圈高星项目后发现，真正该先搞懂的是 5 类能力。

## Scene 04, 16-22s

流程、证据、分工、实验、应用。

## Scene 05, 22-29s

LangGraph 解决的是下一步谁做，状态放哪里，失败怎么重试。

## Scene 06, 29-36s

LlamaIndex 解决的是资料从哪里来，回答能不能追溯。

## Scene 07, 36-43s

CrewAI 把复杂工作拆成多个角色。

## Scene 08, 43-50s

AutoGen 和 AG2 适合观察多个 Agent 怎么对话、协商和调用。

## Scene 09, 50-58s

Dify 和 OpenAI Agents SDK，更接近把 Agent 做成能用的应用。

## Scene 10, 58-68s

这也是 Piko 的路线。不训练模型，不从零造 runtime，用成熟框架，加上证据链和验证。

## Scene 11, 68-75s

真正能落地的 Agent，不是一个工具，是一条能被验证的生产线。
