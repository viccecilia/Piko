## 小红书正文

最近看 GitHub 高星 AI 项目，我发现很多人学 Agent 的方式太累了：今天收藏一个框架，明天又换一个新项目，最后越学越散。

今天这个项目是 LlamaIndex。它经常被放在 RAG 或知识库里讲，但我觉得它真正值得普通人学的，不是“又一个框架”，而是让 AI 回答能回到资料来源。

很多 AI 应用不是不会回答，而是回答完以后说不清：这句话来自哪里？资料是什么版本？有没有原文可以复查？

如果你正在做知识库、文档问答、研究助手、内容资料整理，建议先记住这条学习地图：

资料进入系统 -> 加元信息 -> 建索引 -> 检索相关片段 -> 回答带来源。

学开源项目，不是把框架搬回来，而是把它解决问题的方式学会。
