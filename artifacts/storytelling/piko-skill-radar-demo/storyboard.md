# 视频分镜

比例：9:16  
时长：约 55 秒  
风格：知识类短视频，深色背景，白字大标题，粉蓝强调色。  
参考素材：`assets/reference-article.png`

## Scene 1 0-5s

画面：参考文章截图向上缓慢滚动，前景大字。

屏幕字：

```text
我给 Piko 加了一个
每日 Skill 雷达
```

口播：我最近给 Piko 加了一个新想法，叫每日 Skill 雷达。

## Scene 2 5-12s

画面：三个问题卡片依次弹出。

屏幕字：

```text
新工具很多
到底换不换？
怎么判断？
```

口播：它解决的不是找工具，而是判断一个工具到底值不值得进入系统。

## Scene 3 12-24s

画面：雷达扫描动画，出现 GitHub、Agent、RAG、Plugin、Guardrail。

屏幕字：

```text
每天扫描
高星项目 / Agent 框架 / RAG / 插件 / 验证工具
```

口播：Piko 会每天扫描高星开源项目、Agent 框架、RAG 工具、插件系统和自动化工具。

## Scene 4 24-36s

画面：评分仪表盘。

屏幕字：

```text
准确性
稳定性
成本
安全
License
```

口播：然后给每个能力打分，看它准不准、稳不稳、贵不贵、安不安全。

## Scene 5 36-47s

画面：流程箭头。

屏幕字：

```text
扫描 -> 提案 -> Worker -> Verify -> 人类确认
```

口播：但它不会自动改代码。真正进入系统，还要经过 Worker 实现、Verify 验证，最后人类确认。

## Scene 6 47-58s

画面：Piko 中心节点，连接游戏、电商、SaaS、旅游。

屏幕字：

```text
目标：
系统自动执行
人类最后确认
```

口播：我的目标是，让系统自己发现任务、选择工具、执行和验证。人类只做最后一步确认。
