# Article Template

Piko articles solve one clear player need. They should sound like a careful editor helping a player, not like generic AI text.

## Style Rules

Do:

- Start with the answer.
- Say what to try first.
- Keep platform and version scope visible.
- Explain risk before risky steps.
- Show source transparency.
- Use short, practical wording.

Do not:

- Use generic intros.
- Pretend Piko personally tested something.
- Copy maps, images, tables, or guide layouts from other sites.
- Mix unrelated intents into one article.
- Recommend unsafe downloads, executable replacement, cheats, exploits, or anti-cheat bypasses.
