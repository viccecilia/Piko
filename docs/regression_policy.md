# Regression Policy

Regression plans describe commands to run after an approved change.

## Default Commands

```powershell
python -m pytest
python -m packages.workflows.article_pipeline
```

## Constraints

- Regression plans must not contain destructive commands.
- Regression plans must not deploy.
- Regression plans must not enable live connectors unless a live-connector round explicitly asks for it.
- Regression commands generated by the self-improvement loop are plans, not automatically executed commands.
